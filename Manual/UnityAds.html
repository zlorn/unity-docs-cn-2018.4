Unity Ads

Unity Ads SDK 由领先的移动游戏引擎创建，无论您是在 Unity、xCode 还是 Android Studio 中进行开发，都能为您的游戏提供全面的变现服务框架。它简化了广告、应用内购买和分析，使用最先进的机器学习来最大化您的收入，同时保持良好的玩家体验。入门非常简单。

本页面将说明使用 Unity Ads 实现收入最大化的过程。您可以逐步跟随操作，或跳转到最适合您所在的开发阶段的集成步骤。

注意：此处引用的许多资源都会重定向到核心 Unity Ads 文档。

要求

Unity 建议始终使用最新的 Ads SDK。此处许多文章中讨论的 API 仅在 SDK 3.0 及更高版本中可用。

准备

如果您是 Unity Ads 的新手，请在实现前遵循以下基本步骤：

1.在开发者控制面板 (Developer Dashboard) 上注册 Unity Developer Network (UDN) 帐户。 2.配置广告位 (Placements) 以接收变现内容。 3.下载并导入最新的 Unity Ads SDK。 4.在开始之前，查看我们的最佳实践指南以更好地了解您的实现策略。

实现

以下实现资源适用于使用 Unity 制作的游戏：

1.使用变现 API 集成 Unity Ads。 2.扩展您的基本广告集成：
i.奖励玩家观看广告。
ii.整合横幅广告。
iii.整合增强现实 (AR) 广告。
3.如果游戏使用应用内购买，请集成应用内购推荐 (IAP Promo)。
i.如果您使用 Unity IAP，请单击此处。
ii.如果您使用自定义的 IAP 实现，请单击此处。
iii.（可选）使用原生推荐 (Native Promo) 资源自定义您的推荐。 4.将广告和推荐广告位转换为个性化广告位 (Personalized Placements) 以利用 Unity 的收入优化功能。

管理、分析、优化

除了实现之外，Unity 还可让您对策略进行微调：

  • Unity 开发者控制面板 (Developer Dashboard) 允许您管理广告实现。使用控制面板的强大指标工具可以借助数据驱动型方法来微调您的变现策略。
  • 了解如何过滤广告以定位目标受众。
  • 不要错过收入；一旦游戏激活，请务必添加应用商店详细信息。
  • 实现标准事件来帮助 Unity 的机器学习模型根据您的游戏经济对变现进行优化。
  • 注册自动付款。

支持

有问题？我们为您提供帮助！以下资源可以帮助解决您的问题：

  • 浏览 Unity Ads 社区论坛。
  • 搜索 Unity Ads 变现知识库。
  • 联系 Unity Ads 支持部门提问。
