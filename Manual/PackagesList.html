包文档

一个包中包含适合项目各种需求的功能。这可能包括随 Editor 一起安装的所有 Unity 核心功能，或者可根据需要安装的其他包。请单击当前可用的以下任意包以查看该包的相关文档。

已发布的包

已使用此版本的 Unity 测试了以下包。要在项目中安装这些包，请选择 Window > Package Manager。

  • Ads（预装在每个项目中）
  • Analytics（预装在每个项目中）
  • Package Manager UI（预装在每个项目中）
  • In App Purchasing（预装在每个项目中）
  • TextMesh Pro（预装在每个项目中）
  • Alembic
  • Asset Bundle Browser
  • Cinemachine
  • Oculus (Android)
  • Oculus (Desktop)
  • OpenVR
  • Post-processing
  • ProBuilder
  • Windows Mixed Reality
  • Xiaomi
